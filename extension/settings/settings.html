Quick Database Deployment Steps
  1. Navigate to Supabase Console and construct a new database project.
  2. Execute the exact schema tables using the provided supabase-schema.sql file inside the Supabase SQL Editor.
  3. Retrieve your project parameters (Project URL + Anon Key) from **Settings → API**.
  4. Deploy your FastAPI project folder directly onto cloud platforms like Railway (free tiers supported).
  5. Paste the endpoints above, hit **Save**, and test the integration!
